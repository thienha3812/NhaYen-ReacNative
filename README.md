***Ứng dụng điều khiển nhà yến***
#### Mô tả : Ứng dụng dùng để giám sát và điều khiển nhà yến.
***Chức năng***
* Điều khiển bật tắt đèn
* Hiển thị lưu đồ
* Giám sát nhà yến qua camera

***Cài đặt***
```
	npm install
	react-native run-android // on android
	react-native run-ios // on ios
```
